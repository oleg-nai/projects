Задача
В  текстовом файле test1.txt находятся данные о студентах : фамилия, имя, отчество, средний бал(с буквой Е, е -это запись в экпоненциальном виде),  идентификационный номер.
 Каждая запись о студенте - в отдельной строке. Разделители : пробел, подчеркивание, точка сзапятой: _;
Разделитель "точку" в вещественных числах , для корректного чтения - можно в файле заменить на "запятую"!
	
Разработать классы (Student,  ClassBD),  разработать итерфейс InterfBD с одним методом и реализовать метод в классе.
Класс Student - содержит информацию об одном студенте, 
класс ClassBD - содержит стандартные коллекции (List и MAP, для МАР ключ- фамилия).

в классах должны быть  методы :
	1)	Перегруженный метод toString для Student

	2)	Метод чтения данных из текстового файла в List

	3)	Метод чтения данных из текстового файла в Map (для МАР ключ- фамилия).

	4)	Метод удаляющий записи с некорректными данными  в среднем балле ( если присутствуют символы:  !/"№%:?*()+)
		и записывающий  в результирующий текстовый файл rezult.txt

	5)	Метод удаляющий записи с некорректными данными(из Map коллекции):: (фамилия с цифрами) 
		и записывающий  в результирующий текстовый файл rezultfam.txt

	6)	Метод cортирующий данные класса LIST по полю имя и среднему баллу(если одинаковвые фамилии, то сортирутся по ср. баллу),
		 использовать Сomparator и записывающий  в результирующий текстовый файл rezultsort.txt

	7)	Сериализация объекта List в ХМL формат и записывающий  в результирующий текстовый файл rezultxml.txt
