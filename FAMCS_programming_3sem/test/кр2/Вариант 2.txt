Задача
В  текстовом файле test2.txt находятся данные о студентах : фамилия, имя, дата рождения(число месяца,месяц,год), 
средний бал , идентификационный номер(с буквой Е, е -это запись в экпоненциальном виде).
Разделитель "точку" в вещественных числах , для корректного чтения - можно в файле заменить на "запятую"!
 Каждая запись о студенте - в отдельной строке. Разделители : пробел, подчёркивание, точка сзапятой: _;
 	
Разработать классы (Student,  ClassBD),   разработать итерфейс InterfBD с одним методом и реализовать метод в классе.
Класс Student - содержит информацию об одном студенте,
 класс ClassBD - стандартные коллекции (List и MAP, для МАР, ключ- идентификационный номер).

в классах должны быть  методы :

	1)	Перегруженный метод toString

	2)	Метод чтения данных из текстового файла в List

	3)	Метод чтения данных из текстового файла в Map (для МАР ключ- идентификационный номер).

	4)	Метод удаляющий записи с некорректными данными в идентификационн номере (из Map коллекции): ( если присутствуют символы:  !/:?*()_+)
		и записывающий  в результирующий текстовый файл rezult.txt

	5)	Метод удаляющий записи с некорректными данными: (дата рождения:)
		и записывающий  в результирующий текстовый файл rezultdata.txt. Использовать классы Data, Calendar

	6)	Метод cортирующий данные класса LIST по полю имя и идетификационному номеру (если одинаковвые имена, то сортирутся по инд. номеру), 
		использовать Сomparator и записывающий  в результирующий текстовый файл rezultsort.txt

	7)	Сериализация объекта класс ClassBD в бинарный вид
		и записывающий  в результирующий текстовый файл rezultbin.txt
