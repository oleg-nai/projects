Выполните следующие действия для добавления библиотек XStream в созданный проект:
1.	Скопируйте Jar в свой каталог libs/
2.	Откройте Настройки Проекта (Ctrl Alt Shift S)
3.	На панели настроек проекта слева выберите пункт модули
4.	На большой правой панели выберите вкладку зависимости
5.	Нажмите кнопку Добавить... кнопка в дальнем правом углу экрана (если у вас есть меньший размер экран, как и я, вам, возможно, придется перетащить изменение размера вправо, чтобы видеть его)
6.	В раскрывающемся списке добавить параметры выберите "Library". Появится диалоговое окно "Choose Libraries".
7.	Нажмите "New Library..."
8.	Выберите подходящее название для библиотеки
9.	Нажмите "Attach Classes..."
10.	Выберите Jar из каталога libs/ и нажмите OK, чтобы закрыть его
